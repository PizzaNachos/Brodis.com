Minecraft Court Plugin!

What did it do?

  • Lets players file suit against each other
  • Selects Jurors, a presiding judge, and lawyers
  • Interacts with the minecraft spigot API to collect court fees and pay out to the "winner"

What did i learn?

  • How to use Java in the context of someone else's framework
  • How to meet and manage the expectations of less technical people that you work with
  • Minecraft Spigot Framework
